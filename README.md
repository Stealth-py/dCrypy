<h1 align="center">dCrypy</h1>
<p align = "center">A CLI based decryption/encryption tool</p>

## Installation
### Through pip
#### For Windows:
- `pip install dCrypy`
#### For Linux/MacOS:
- `pip3 install dCrypy`
